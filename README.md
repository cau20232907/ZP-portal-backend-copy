ZPserver 프로젝트 백엔드 코드의 복사본입니다.<br/>
원본이 배포 문제로 private으로 설정되어 있어, 현재 코드를 따로 복사해 올려놓게 되었습니다.<br/>
따라서 갱신이 빠르지 않을 수 있습니다.<br/>
설정 파일에 적혀 있는 사이트 URL은 SSO 아키텍쳐 서버 주소입니다.<br/>
SSO 로그인 아키텍쳐는 이미 서버에 띄워져 있어, 이를 이용해 작업했습니다.
